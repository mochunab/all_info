## 1. 3C 분석

|------|------|
| **서비스명** | 아카인포 |
| **핵심 기술** | 9단계 자동 감지 크롤링 파이프라인 + AI 요약 (Gemini 2.5 Flash Lite) |
| **주요 기능** | 50+ 소스 자동 크롤링, 1줄 AI 요약 + 태그, AI 면접 코칭 채팅 |
| **기술 스택** | Next.js 14, Supabase, Vercel, Gemini/OpenAI |
| **차별화 역량** | 임의의 URL을 등록하면 크롤러 타입을 자동 판별하는 범용 AI 크롤러 |
| **현재 단계** | MVP 출시 완료, 프로덕션 운영 중 |
| **수익 모델** | 미정 (현재 무료 제공) |

**핵심 자산**
- 9종 크롤러 전략 (STATIC, SPA, RSS, Sitemap, 플랫폼별 등) + 자동 복구 체인
- AI 요약 파이프라인 (Edge Function 우선 → 로컬 OpenAI 폴백)
- 멀티유저 피드 아키텍처 (홈피드/마이피드 분리, user_id 스코핑)

| 항목 | 내용 |
|------|------|
| **규모** | 한국 연간 구직자 약 60만 명 (통계청 기준), 취준 기간 평균 11개월 |

| **Pain Point** | 매일 2시간 뉴스 탐색, "최근 트렌드?" 질문에 멈칫, 스크랩 무덤 |
| **지불 의향** | 월 5,000~10,000원 (취준 비용 대비 낮은 가격 선호) |

**2차 타겟: 주니어 직장인 (1~3년차)**
- 업계 동향 파악이 필요하지만 시간이 부족한 직장인
- 이직 준비 시 업계 지식 업데이트 수요

**3차 타겟: B2B (커리어 플랫폼, 취업 교육 기관)**
- 화이트라벨 콘텐츠 큐레이션 솔루션
- 채용 담당자용 업계 트렌드 리포트

| **수익 모델** | B2B (기업 채용 서비스) | 무료 (초기 단계, 앱 리뷰 없음) | 미정 (프리미엄 전환 예정) |
| **플랫폼** | 웹 + 크롬 확장 | iOS 앱 + 웹 | 웹 (모바일 반응형) |

#### 아카인포 차별화 요약

| 차별화 포인트 | 경쟁사 대비 |
|--------------|-----------|
| **AI 자동화 End-to-End** | 서핏/크루마는 사람이 골라주는 콘텐츠. 아카인포는 수집→요약→면접 코칭까지 AI 자동 |
| **전 업종 커버** | 서핏(IT 5개 직무), 크루마(IT 한정). 비IT 취준생을 커버하는 유일한 서비스 |
| **콘텐츠→면접 답변 원스톱** | 크루마는 면접 질문 DB와 뉴스가 분리. 아카인포는 기사 기반 실시간 AI 코칭으로 통합 |
| **사용자 주도 소스 관리** | 내가 가고 싶은 회사 블로그를 직접 등록하면 매일 자동 수집 |
| **9단계 범용 AI 크롤러** | 임의 URL → 자동 전략 감지 → 자동 복구. 수동 큐레이션 대비 확장성 압도적 |

**경쟁 공백 (White Space)**

기존 서비스 중 **"실시간 업계 콘텐츠 자동 수집 + AI 요약 + 면접 답변 코칭"**을 하나의 플랫폼에서 제공하는 곳이 없음. 서핏은 IT 재직자 중심이고 면접 기능이 없음. 크루마는 면접 기능이 있으나 뉴스와 분리되어 있고 IT 한정. 아카인포만이 **전 업종 콘텐츠 자동 수집과 면접 코칭을 실시간 연결**하는 유일한 포지션.

**SO 전략 (강점 × 기회)**: 자동 크롤링 + AI 코칭 기술력을 무기로 취준생 시장 선점 → B2B 확장
**WO 전략 (약점 × 기회)**: 취준 커뮤니티 바이럴로 인지도 확보 → 프리미엄 전환으로 수익화
**ST 전략 (강점 × 위협)**: 범용 크롤러 자동 복구 체인으로 차단 대응 → AI 차별화 유지
**WT 전략 (약점 × 위협)**: 저작권 안전한 요약/링크 모델 유지 → 최소 비용 운영 구조

## 3. STP 분석

### Segmentation (시장 세분화)

| 세그먼트 | 규모 (추정) | 특성 | 지불 의향 |
|----------|-------------|------|-----------|
| **A. 취업 준비생** | 60만/년 | 시간 부족, 정보 불안, 가격 민감 | 월 5,000~10,000원 |
| **B. 이직 준비 직장인** | 120만/년 | 시간 극히 부족, 업계 감각 유지 필요 | 월 10,000~15,000원 |
| **C. 주니어 직장인 (자기계발)** | 200만+ | 업무 관련 트렌드 파악, 회의 준비 | 월 10,000~20,000원 |
| **D. 커리어 플랫폼 (B2B)** | 50+ 기업 | 콘텐츠 큐레이션 니즈, API 연동 | 월 50만~300만원 |

### Targeting (목표 시장)

**1차 타겟: 세그먼트 A (취업 준비생)**

선정 이유:
- 가장 절실한 Pain Point ("면접에서 할 말이 없다")
- 입소문 전파력 높음 (에브리타임, 오픈카톡 등 커뮤니티 활발)
- 취업 성공 시 자연스러운 세그먼트 B/C 전환
- 무료 → 유료 전환 시 취업 축하 할인 등 감성 마케팅 가능

**2차 타겟: 세그먼트 B (이직 준비 직장인)**

진입 시점: 1차 타겟에서 PMF 확인 후 (MAU 1만+ 달성 시)

### Positioning (포지셔닝)

**포지셔닝 맵**

```
             전문적/깊이 있는
                  ↑
                  │
    뉴닉/어피티   │   ★ 아카인포
    (해설형)      │   (자동 큐레이션 + AI 코칭)
                  │
  수동 ──────────┼────────────── 자동
  (사용자 탐색)   │              (AI 자동화)
                  │
    네이버 뉴스   │   Feedly
    (검색형)      │   (RSS 구독형)
                  │
                  ↓
             대중적/얕은
```

**포지셔닝 스테이트먼트**

> 면접 준비에 불안한 취준생을 위해,
> 아카인포는 매일 자동으로 업계 콘텐츠를 수집하고 AI가 면접 답변까지 코칭해주는
> 유일한 면접 준비 플랫폼입니다.
> 뉴스 검색이나 뉴스레터와 달리, 수동 탐색 없이 30초 만에 면접 준비가 완료됩니다.

## 4. 핵심 컨셉

> **"30년이 달라진다, 하루 30초로"**

- 30초(하루 투자) ↔ 30년(커리어 전체)의 숫자 대칭으로 작은 습관이 만드는 큰 변화를 압축
- AI가 매일 업계 콘텐츠를 수집·요약하고, 면접에서 어떻게 말할지까지 코칭해주는 자동화 면접 준비 도구

## 5. 브랜드 슬로건

> **메인: "30년이 달라진다, 하루 30초로"**
> **서브: "나만의 면접 치트키"**

**슬로건 설계 의도**
- "30년 vs 30초" → 시간 대비 효과 (행동경제학 Anchoring + Loss Aversion)
- "달라진다" → 변화의 기대감, 현재와 미래의 갭 자극
- "면접 치트키" → 비밀 무기 이미지, 취준생이 직관적으로 공감하는 게임 용어 차용
- "나만의" → 개인화 가치, 나에게 맞춤화된 도구임을 강조

**서브 카피 (상황별)**
- 광고용: "AI가 매일 업계 브리핑을 읽고, 나만의 면접 치트키를 만들어드립니다"
- SNS용: "30년이 달라진다, 하루 30초로"
- 앱스토어용: "나만의 면접 치트키 — AI 업계 브리핑으로 자신감이 달라집니다"

## 8. B2B 사업 확장 검토: 유학생-지방 중소기업 매칭 서비스

> **가설**: 취업 정보 불편을 가장 크게 겪는 유학생과, 서울 쏠림으로 인재 유치가 어려운 지방 중소기업을 중개하는 B2B 서비스

### 8.1 시장 기회 — 유학생 측

#### 핵심 수치

| 지표 | 수치 | 출처 |
|------|------|------|
| 총 유학생 수 | **25.3만 명** (2025) | [1] |
| 한국 취업 희망 비율 | **86.5%** | [2] |
| 실제 취업률 | **13.8%** (졸업자 대비) | [3] |
| 비수도권 유학생 비율 | **45.8%** (11.6만 명) | [1] |
| 현 거주 지역(비수도권) 취업 희망 | **55.1%** | [4] |
| 연봉/복지 충족 시 비선호 지역 OK | **74.4%** | [5] |
| 서울 소재 대학 → 서울 취업 고집 | **95.9%** | [6] |
| 대기업/선호직무 고집 안 함 | **~60%** | [7] |
| E-9으로라도 중소기업 근무 의향 | **58.8%** | [2] |
| 비자 만료 이탈률 | **42.9%** | [3] |

#### 구조적 갭: 희망 86.5% vs 실제 취업 13.8%

유학생의 절대다수가 한국 취업을 희망하지만, 실제 취업에 성공하는 비율은 13.8%에 불과하다. 이 거대한 갭의 원인은:

- **정보 비대칭**: 채용 공고에 외국인 지원 가능 여부/비자 지원 여부 미표기
- **비자 전환 장벽**: D-2 → E-7 전환 과정의 복잡성 (유학생 66.7%가 "어렵다" 응답 [8])
- **네트워크 부재**: 유학생의 79.4%가 지인/커뮤니티를 통해 근로 정보 획득 — 공식 채널 부재 [9]

#### 핵심 인사이트: 타겟은 "지방대 유학생"

- 서울 소재 대학 유학생은 95.9%가 서울을 고집 → 설득 비용이 매우 높음
- **지방대 재학/졸업 유학생(11.6만 명)**은 이미 지방에 적응했고, 55.1%가 현지 취업 희망
- 조건부 수용(74.4%)의 조건 = **급여/복지** → 가격이 아닌 매칭 품질이 핵심

### 8.2 시장 기회 — 지방 중소기업 측

#### 핵심 수치

| 지표 | 수치 | 출처 |
|------|------|------|
| 외국인 고용 유지/확대 계획 | **98.2%** (1,223개사) | [10] |
| 향후 5년 외국인 채용 확대 | **65.3%** (98개 지역기업) | [7] |
| 일반 외국인보다 유학생 선호 | **51.3%** (일반 34%) | [11] |
| 외국인 고용 사유: 내국인 구인난 | **82.6%** | [12] |
| 비수도권 → 수도권 이동 청년(15~39세) | **13.1만 명 (69.6%)** | [13] |
| 비수도권 기업 최대 격차 요인: 인력 확보 | **66.2%** | [14] |
| 산업기술인력 부족 인원 | **3.9만 명** (부족률 2.2%) | [15] |

#### 기업 측 최대 장벽

| 순위 | 장벽 | 비율 | 출처 |
|------|------|------|------|
| 1 | 의사소통/한국어 수준 | **50~52%** | [10][16] |
| 2 | 짧은 체류 허용 기간 | **47.4%** | [16] |
| 3 | 법령/제도 이해 부족 | **49.5%** | [17] |
| 4 | 복잡한 채용 절차 | **36.9%** | [16] |
| 5 | 조직문화/팀워크 적응 | **27~40%** | [16][17] |

채용 후 가장 필요한 지원 서비스: **"비자 및 체류 관리(연장 등)"** — 64% [17]

#### 핵심 인사이트: 간절하지만 스스로 못 한다

- 수요는 압도적 (98% 유지/확대, 유학생을 일반 외국인 대비 1.5배 선호)
- 그러나 비자 절차를 모르고(49.5%), 소통이 걱정되고(52%), 법령 이해가 부족
- → **"비자 행정 대행 + 한국어 능력 검증 + 매칭"을 원스톱 제공하면 지불 의향 높음**

### 8.3 비자 구조 비교 (E-9 vs E-7)

| 항목 | E-9 (비전문취업/고용허가제) | E-7 (특정활동/전문인력) |
|------|--------------------------|----------------------|
| **대상** | 비숙련~반숙련 노동 | 91개 직종 전문인력 |
| **채용 자율성** | 낮음 (정부 배정) | 높음 (기업 직접 채용) |
| **임금** | 최저임금 의무 | 연 **3,112만원+** (2026 GNI 기준) [18] |
| **보험 부담** | **높음** (월 임금 8.3% 적립 등) | **낮음** (의무 없음) |
| **체류기간** | 최대 4년 10개월 | 1~3년 (갱신 가능) |
| **행정 복잡도** | 중간 (정부 시스템) | **높음** (직접 서류 준비) |
| **유학생 경로** | D-2 → E-9 전환 가능 | D-2 → D-10 → E-7 전환 |

**사업 기회**: E-7 비자의 높은 행정 복잡도가 기업과 유학생 양측의 진입 장벽 → 이를 대행하는 서비스에 대한 수요 존재

- **비자 행정 + 채용 + 정착 지원**을 원스톱으로 제공하는 서비스는 전무

**왜 이 구조가 유효한가:**

1. **콜드 스타트 해소**: 아카인포가 무료 콘텐츠 도구로 유학생을 먼저 모은다. 투사이드 마켓의 공급 측(유학생)을 별도 제품으로 확보 → 기업에 "이미 N명의 유학생 인재 풀이 있다"고 제안 가능
2. **유학생 CAC 최소화**: 콘텐츠 크롤링 + AI 면접 코칭이라는 실사용 가치가 있으므로 광고비 없이 자연 유입 가능 (취준 커뮤니티 바이럴)
3. **자연스러운 프로필 전환**: 아카인포에서 콘텐츠를 소비하던 유학생이 → 이력서 등록 → 매칭 대상 인재 풀로 전환 (별도 가입 없이)
4. **기존 기술 100% 재활용**: 크롤러, AI 요약, 멀티유저 아키텍처 모두 그대로 활용

#### 수익 구조

| 항목 | 과금 대상 | 가격대 | 비고 |
|------|-----------|--------|------|
| **아카인포 기본** | 유학생 | **무료** | 콘텐츠 피드, 기본 AI 코칭 |
| **아카인포 프리미엄** | 유학생 | **월 2,900~4,900원** | 무제한 AI 코칭, 고급 면접 분석 (최소 수익) |
| **채용 성사 수수료** | 기업 | **연봉의 8~15%** (250~470만원) | 핵심 수익 — 매칭 성사 시 과금 |
| **비자 행정 대행** | 기업 | **건당 50~100만원** | E-7 서류 준비, 출입국 신청 대행 |

**BEP 시나리오:**
- 아카인포 운영비: 월 ~5만원 (현행 유지)
- 매칭 1건 성사 수수료 (연봉 3,100만원 × 10%): **~310만원**
- **월 1건만 성사해도 흑자** → 비자 대행, 구독료는 추가 수익

**SO 전략**: 아카인포 무료 퍼널로 유학생 풀 선점 → 정부 정책(F-2-R, RISE) 활용하여 지방 기업 B2B 영업

**WT 전략**: Phase 0에서 수동 매칭으로 WTP 검증 후 플랫폼 투자 → 리스크 최소화

```
             유학생 측                        기업 측

             ❌ 비자 전환 어려움 66.7%          ❌ 소통 장벽 52%
             ❌ 채용 정보 접근 불가              ❌ 채용 경로 없음

경쟁사?      ❌ 지방 중소기업 특화 매칭 서비스 전무
             ❌ 비자 행정 + 채용 + 정착 원스톱 없음
             ❌ 정부 시도 중이나 아직 미구현

우리 해법?   ✅ 아카인포 = 무료 퍼널 → 유학생 인재 풀 확보 (콜드 스타트 해소)
             ✅ B2B 매칭 = 기업 과금 → 매칭 1건(~310만원)으로 즉시 흑자
             ✅ 기존 기술 100% 재활용 (크롤러, AI 요약, 멀티유저)
```

**양측 다 원하는데 연결이 안 되고 있다. 그리고 이걸 해주는 서비스가 없다.**

데이터 기반으로 사업 가설은 충분히 뒷받침된다. 통합 모델의 핵심 구조:

- **아카인포 (퍼널)**: 무료/최소 요금으로 유학생을 모은다 → CAC 최소화
- **B2B 매칭 (수익)**: 기업에 인재 풀 + 비자 대행 + 온보딩을 팔아 돈을 번다
- **BEP**: 월 매칭 1건(~310만원) = 운영비(~5만원) 대비 **즉시 흑자**

남은 핵심 변수:
1. **비자 행정 전문성 확보** — 행정사 파트너십 필수
2. **첫 매칭 성사** — Phase 0~1에서 수동으로라도 1건 증명
3. **유학생 → 프로필 전환율** — 아카인포 사용자가 이력서를 등록하는 비율
