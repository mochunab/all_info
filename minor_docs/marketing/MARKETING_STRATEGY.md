## 2. 채널별 실행 전략

### 2-1. 에브리타임 (최우선, CAC ≈ 0원)

> 취준생 집중도 1위 채널. 자교 게시판 특성상 "선배의 꿀팁" 프레이밍이 핵심.

**전술**

| 액션 | 실행 방법 | 빈도 |
|------|-----------|------|
| **꿀팁 포스트** | "면접에서 최근 트렌드 질문 받았는데 이거 쓰니까 바로 합격함" → 아카인포 캡처 첨부 | 주 2~3회 |
| **면접 후기 연계** | "XX 기업 면접 다녀왔는데 업계 트렌드 질문 나옴 → 이걸로 준비했음" | 면접 시즌 집중 |
| **시험 기간 피하기** | 중간/기말 2주 전 ~ 1주 후 포스팅 중단 | 학사 일정 연동 |

**포스트 템플릿**

```
[꿀팁] 면접에서 "최근 마케팅 트렌드 아는 거 있어요?" 나왔을 때

나도 처음엔 뉴스 찾아보다 포기했는데
아카인포라는 사이트 쓰고 나서 30초면 됨

매일 자동으로 업계 기사 수집해주고
AI가 1줄로 요약해줌 + 면접 답변도 만들어줌

나는 "리테일 미디어 네트워크 성장" 이거 말했더니
면접관이 끄덕끄덕하더라

무료니까 써보셈
[링크]
```

**핵심 원칙**
- 광고 느낌 절대 금지 — "나도 써봤는데" 톤
- 구체적인 면접 질문 + 답변 예시 포함 → 가치를 먼저 제공
- 링크는 본문 마지막에 자연스럽게 배치

### 2-2. 취업 오픈카톡 / 디스코드

> 업종별·기업별 오픈카톡방(50~500명)은 높은 참여율과 신뢰도를 보유. "정보 공유자" 포지션 확보가 핵심.

**전술**

| 액션 | 설명 |
|------|------|
| **데일리 브리핑 공유** | 매일 09:30 아카인포 홈피드 기사 중 해당 업종 기사 1~2개 캡처 + 링크 공유 |
| **면접 질문 연결** | "XX 기업 면접에서 이 주제 나올 수 있음 → 아카인포에서 AI 코칭 받아보세요" |
| **방장에게 DM** | "매일 업계 기사 요약해서 공유해도 될까요?" → 허락 받고 공식 루틴화 |

**타겟 카톡방 우선순위**
1. 마케팅 직무 취준 (규모 大, 트렌드 수요 높음)
2. IT/스타트업 취준 (얼리어답터 비율 높음)
3. 금융/컨설팅 취준 (업계 지식 중요도 극대)
4. 유통/리테일 취준 (아카인포 기본 카테고리와 일치)

## 3. 바이럴 루프 설계

### 3-1. 자연 바이럴 트리거

```
사용자 A가 아카인포로 면접 준비
    ↓
면접 합격
    ↓
에브리타임/카톡에 "이거 써서 합격했다" 후기 공유 (자발적)
    ↓
친구 B, C, D 유입
    ↓
반복
```

**바이럴 계수(K) 목표: 1.2+**

바이럴을 촉진하는 제품 내 장치:

| 장치 | 설명 | 구현 난이도 |
|------|------|------------|
| **공유 카드** | AI 코칭 결과를 이미지 카드로 생성 (인스타 스토리 최적 비율) | 중 |
| **합격 배지** | "아카인포로 XX 기업 합격" 공유용 이미지 | 낮음 |
| **초대 코드** | 친구 초대 시 프리미엄 7일 무료 체험 | 중 |
| **주간 브리핑 공유** | "이번 주 내 업계 핵심 뉴스 3개" → 1클릭 카톡/인스타 공유 | 낮음 |

## 4. 런칭 캠페인: "30초 챌린지"

### 컨셉

> **"30일 30초 면접 준비 챌린지"**
> 매일 아카인포를 30초씩 확인하고, 30일 뒤 면접 자신감이 어떻게 달라졌는지 공유하는 챌린지.

### 실행 계획

| 단계 | 기간 | 액션 |
|------|------|------|
| **사전 모집** | D-14 ~ D-1 | 에브리타임/인스타에서 참가자 50명 모집. "30일 뒤 면접 자신감 변화 측정" |
| **챌린지 기간** | D-Day ~ D+30 | 매일 참가자에게 카톡/이메일로 "오늘의 30초 브리핑" 푸시. 주 1회 "이번 주 배운 것" 공유 독려 |
| **결과 공유** | D+31 ~ D+37 | 참가자 비포/애프터 자신감 점수 비교. 후기 콘텐츠화. "면접 자신감 78% 향상" 등 수치 강조 |

### 챌린지 설문 (비포/애프터)

```
Q1. 면접에서 "최근 업계 트렌드" 질문이 나오면? (1~10점)
Q2. 관심 업종의 최근 이슈를 3개 이상 말할 수 있는가? (예/아니오)
Q3. 면접 답변에 구체적인 사례를 포함할 수 있는가? (1~10점)
Q4. 면접 준비에 하루 몇 분 투자하는가?
```

## 6. 유학생 획득 전략 (1차 타겟, 최우선)

> **왜 유학생이 1차 타겟인가**: 유학생은 취업 정보 불편을 가장 크게 겪는 세그먼트이자, 향후 B2B 매칭 서비스의 공급 측(인재 풀)이 된다. 아카인포를 무료/최소 요금으로 제공하여 유학생을 모으고, 이후 지방 중소기업에 인재 풀로 연결하는 것이 전체 비즈니스의 핵심 구조다. (상세: BUSINESS_STRATEGY.md 8장)

### 6-0. 유학생 시장 핵심 수치

| 지표 | 수치 |
|------|------|
| 총 유학생 수 | **25.3만 명** (2025) |
| 한국 취업 희망 | **86.5%** (약 21.9만 명) |
| 실제 취업률 | **13.8%** → 거대한 갭 = 아카인포 기회 |
| 비수도권 유학생 | **45.8%** (11.6만 명) |
| 정보 획득 경로 | **79.4%**가 지인/커뮤니티 → 커뮤니티 마케팅 필수 |

### 6-1. 국적별 채널 전략

| 국적 | 규모 (추정) | 1차 채널 | 2차 채널 | 메시지 핵심 |
|------|------------|---------|---------|------------|
| **베트남** | ~6.5만 | Facebook 유학생 그룹 | Zalo 단톡 | 무료 AI 면접 코칭, 한국 취업 정보 자동 수집 |
| **중국** | ~5.5만 | WeChat 유학생 그룹 | 小红书 (샤오홍슈) | 무료 업계 트렌드 + 모국어 번역 |
| **우즈베키스탄** | ~2만 | Telegram 그룹 | 대학 커뮤니티 | 한국 취업 준비 무료 도구 |

**우선순위: 베트남 > 중국 > 우즈베키스탄 > 나머지** (규모 + 커뮤니티 접근성 기준)

### 6-2. 제품-메시지 맞춤 (유학생 Pain Point 직격)

유학생이 한국 취업에서 겪는 3대 문제와 아카인포의 해결:

| 유학생 Pain Point | 아카인포 해결 | 메시지 |
|-------------------|-------------|--------|
| **한국 업계 정보를 어디서 찾는지 모름** | 50+ 소스 자동 크롤링 + 모국어 번역 (DeepL) | "한국 업계 뉴스, 매일 자동으로 모국어로 받아보세요" |
| **면접에서 한국어로 답변 못 함** | AI 면접 코칭 (기사 기반 답변 생성) | "AI가 면접 답변을 만들어줍니다. 한국어 답변 연습하세요" |
| **어떤 기업/업종을 지원해야 할지 모름** | AI 소스 추천 + 업종별 카테고리 | "관심 업종 선택하면 매일 관련 기사가 옵니다" |
