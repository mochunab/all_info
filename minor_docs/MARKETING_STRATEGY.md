## 2. 채널별 실행 전략

### 2-1. 에브리타임 (최우선, CAC ≈ 0원)

> 취준생 집중도 1위 채널. 자교 게시판 특성상 "선배의 꿀팁" 프레이밍이 핵심.

**전술**

| 액션 | 실행 방법 | 빈도 |
|------|-----------|------|
| **꿀팁 포스트** | "면접에서 최근 트렌드 질문 받았는데 이거 쓰니까 바로 합격함" → 아카인포 캡처 첨부 | 주 2~3회 |
| **면접 후기 연계** | "XX 기업 면접 다녀왔는데 업계 트렌드 질문 나옴 → 이걸로 준비했음" | 면접 시즌 집중 |
| **시험 기간 피하기** | 중간/기말 2주 전 ~ 1주 후 포스팅 중단 | 학사 일정 연동 |

**포스트 템플릿**

```
[꿀팁] 면접에서 "최근 마케팅 트렌드 아는 거 있어요?" 나왔을 때

나도 처음엔 뉴스 찾아보다 포기했는데
아카인포라는 사이트 쓰고 나서 30초면 됨

매일 자동으로 업계 기사 수집해주고
AI가 1줄로 요약해줌 + 면접 답변도 만들어줌

나는 "리테일 미디어 네트워크 성장" 이거 말했더니
면접관이 끄덕끄덕하더라

무료니까 써보셈
[링크]
```

**핵심 원칙**
- 광고 느낌 절대 금지 — "나도 써봤는데" 톤
- 구체적인 면접 질문 + 답변 예시 포함 → 가치를 먼저 제공
- 링크는 본문 마지막에 자연스럽게 배치

### 2-2. 취업 오픈카톡 / 디스코드

> 업종별·기업별 오픈카톡방(50~500명)은 높은 참여율과 신뢰도를 보유. "정보 공유자" 포지션 확보가 핵심.

**전술**

| 액션 | 설명 |
|------|------|
| **데일리 브리핑 공유** | 매일 09:30 아카인포 홈피드 기사 중 해당 업종 기사 1~2개 캡처 + 링크 공유 |
| **면접 질문 연결** | "XX 기업 면접에서 이 주제 나올 수 있음 → 아카인포에서 AI 코칭 받아보세요" |
| **방장에게 DM** | "매일 업계 기사 요약해서 공유해도 될까요?" → 허락 받고 공식 루틴화 |

**타겟 카톡방 우선순위**
1. 마케팅 직무 취준 (규모 大, 트렌드 수요 높음)
2. IT/스타트업 취준 (얼리어답터 비율 높음)
3. 금융/컨설팅 취준 (업계 지식 중요도 극대)
4. 유통/리테일 취준 (아카인포 기본 카테고리와 일치)

### 2-3. 인스타그램 (비주얼 바이럴)

> 릴스/카드뉴스로 "오늘의 업계 인사이트" 시리즈 운영. 바이오 링크 → 아카인포.

**콘텐츠 유형**

| 유형 | 포맷 | 빈도 | 예시 |
|------|------|------|------|
| **오늘의 인사이트** | 카드뉴스 (3~5장) | 주 3회 | "오늘 마케터가 알아야 할 것 3가지" |
| **면접 치트키** | 릴스 30초 | 주 2회 | "면접에서 이 질문 나오면 이렇게 답하세요" |
| **비포/애프터** | 카드뉴스 | 주 1회 | "아카인포 쓰기 전 vs 후 면접 답변 비교" |
| **취준 공감** | 카드뉴스/밈 | 주 1~2회 | "면접에서 최근 트렌드 물어볼 때 표정.jpg" |

**카드뉴스 구조 (오늘의 인사이트)**
```
1장: 후킹 — "마케터 면접에서 이거 모르면 탈락합니다"
2장: 인사이트 요약 — 아카인포 AI 요약 기반
3장: 면접 답변 예시 — AI 코칭 기반
4장: CTA — "매일 30초, 아카인포에서 자동으로 받아보세요 (링크 인 바이오)"
```

**해시태그 세트**
```
#면접준비 #취준생 #면접꿀팁 #취업준비 #면접답변
#마케팅트렌드 #IT트렌드 #비즈니스인사이트
#취준일기 #면접합격 #아카인포
```

## 3. 바이럴 루프 설계

### 3-1. 자연 바이럴 트리거

```
사용자 A가 아카인포로 면접 준비
    ↓
면접 합격
    ↓
에브리타임/카톡에 "이거 써서 합격했다" 후기 공유 (자발적)
    ↓
친구 B, C, D 유입
    ↓
반복
```

**바이럴 계수(K) 목표: 1.2+**

바이럴을 촉진하는 제품 내 장치:

| 장치 | 설명 | 구현 난이도 |
|------|------|------------|
| **공유 카드** | AI 코칭 결과를 이미지 카드로 생성 (인스타 스토리 최적 비율) | 중 |
| **합격 배지** | "아카인포로 XX 기업 합격" 공유용 이미지 | 낮음 |
| **초대 코드** | 친구 초대 시 프리미엄 7일 무료 체험 | 중 |
| **주간 브리핑 공유** | "이번 주 내 업계 핵심 뉴스 3개" → 1클릭 카톡/인스타 공유 | 낮음 |

## 4. 런칭 캠페인: "30초 챌린지"

### 컨셉

> **"30일 30초 면접 준비 챌린지"**
> 매일 아카인포를 30초씩 확인하고, 30일 뒤 면접 자신감이 어떻게 달라졌는지 공유하는 챌린지.

### 실행 계획

| 단계 | 기간 | 액션 |
|------|------|------|
| **사전 모집** | D-14 ~ D-1 | 에브리타임/인스타에서 참가자 50명 모집. "30일 뒤 면접 자신감 변화 측정" |
| **챌린지 기간** | D-Day ~ D+30 | 매일 참가자에게 카톡/이메일로 "오늘의 30초 브리핑" 푸시. 주 1회 "이번 주 배운 것" 공유 독려 |
| **결과 공유** | D+31 ~ D+37 | 참가자 비포/애프터 자신감 점수 비교. 후기 콘텐츠화. "면접 자신감 78% 향상" 등 수치 강조 |

### 챌린지 설문 (비포/애프터)

```
Q1. 면접에서 "최근 업계 트렌드" 질문이 나오면? (1~10점)
Q2. 관심 업종의 최근 이슈를 3개 이상 말할 수 있는가? (예/아니오)
Q3. 면접 답변에 구체적인 사례를 포함할 수 있는가? (1~10점)
Q4. 면접 준비에 하루 몇 분 투자하는가?
```

## 6. 다국어 마케팅 (vi/en/zh/ja)

> 아카인포는 5개 언어(ko/en/vi/zh/ja) UI + 카테고리 동적 번역(DeepL)을 지원.
> 한국 내 유학생(25.3만 명)이 1차 글로벌 타겟.

### 6-1. 유학생 타겟 채널

| 언어 | 타겟 | 채널 | 메시지 |
|------|------|------|--------|
| **vi** | 베트남 유학생 (6.5만) | Facebook 유학생 그룹, 잔트이 (Zalo) | "Phỏng vấn Hàn Quốc - AI coaching miễn phí" |
| **zh** | 중국 유학생 (5.5만) | WeChat 유학생 그룹, 小红书 | "韩国面试准备 AI工具 - 免费" |
| **en** | 영어권 유학생 | Reddit r/korea, Facebook | "AI-powered Korean job interview prep" |
| **ja** | 일본 유학생 | Twitter/X, LINE 그룹 | "韓国就活AI面接コーチング - 無料" |

### 6-2. 제품-메시지 맞춤

유학생에게 아카인포의 가치:
1. **한국 업계 트렌드를 모국어 번역으로 파악** — 카테고리 동적 번역 (DeepL)
2. **AI 면접 코칭으로 한국어 답변 연습** — 기사 기반 면접 답변 생성
3. **나만의 소스 등록** — 모국 기업/업계 소스도 등록 가능 (범용 크롤러)

### 6-3. 유학생 마케팅 실행

| 액션 | 설명 | 우선순위 |
|------|------|---------|
| 랜딩 페이지 다국어 | `/landing` 언어별 메타 태그 + i18n 콘텐츠 | 높음 |
| 대학 국제처 제휴 | 지방대 국제처에 "유학생 취업 지원 도구" 소개 | 중 |
| 유학생 커뮤니티 포스팅 | Facebook/WeChat 그룹에 모국어 사용법 가이드 | 높음 |
| 유학생 앰배서더 | 대학별 1명, 월 5만원 상당 혜택 | Phase 2 이후 |

## 8. 측정 지표 (Marketing Dashboard)

### 8-1. 퍼널 지표

```
인지 (Awareness)
  │ 측정: 웹사이트 방문자 수, 인스타 도달, 에브리타임 포스트 조회
  ↓
획득 (Acquisition)
  │ 측정: 회원가입 수, 소스 등록 수 (비회원 포함)
  ↓
활성화 (Activation)
  │ 측정: 첫 AI 코칭 사용, 첫 소스 등록 → "Aha Moment"
  ↓
리텐션 (Retention)
  │ 측정: D1/D7/D30 리텐션, WAU
  ↓
추천 (Referral)
  │ 측정: 공유 카드 생성 수, 초대 코드 사용 수
  ↓
수익 (Revenue)
  │ 측정: 유료 전환율, MRR, LTV
```

### 8-2. 채널별 핵심 지표

| 채널 | 핵심 지표 | Phase 1 목표 |
|------|-----------|-------------|
| 에브리타임 | 포스트당 유입 수, 가입 전환율 | 포스트당 50+ 클릭 |
| 인스타그램 | 팔로워 수, 릴스 도달, 바이오 클릭 | 팔로워 500+, 릴스당 1,000+ 도달 |
| 블로그 SEO | 키워드 순위, 오가닉 유입 | 상위 20위 키워드 3개+ |
| 오픈카톡 | 공유 후 유입 수 | 방당 월 10+ 유입 |

### 8-3. North Star Metric

> **주간 AI 코칭 사용 횟수 (Weekly AI Coaching Sessions)**

이유:
- AI 코칭 사용 = 아카인포의 핵심 가치 체험 = 리텐션의 선행 지표
- 코칭 결과 공유 = 바이럴 트리거
- 코칭 빈도 높은 유저 = 유료 전환 후보

## 10. 실행 로드맵

### 1개월차: 기반 구축 + 시드 유저

```
Week 1:
  ├─ 인스타 계정 생성 + 프로필 세팅 + 첫 포스트 5개 준비
  ├─ 네이버 블로그 생성 + 첫 SEO 포스트 2개
  └─ 에브리타임 첫 포스트 (3개 대학)

Week 2:
  ├─ 에브리타임 주 3회 포스팅 시작
  ├─ 오픈카톡 3개방 진입 + 데일리 브리핑 공유 시작
  └─ 인스타 카드뉴스 주 3회 시작

Week 3:
  ├─ "30초 챌린지" 사전 모집 시작
  ├─ 블로그 SEO 포스트 2개 추가
  └─ 채널별 성과 측정 + 최적화

Week 4:
  ├─ "30초 챌린지" 시작 (50명)
  ├─ 첫 유학생 커뮤니티 포스팅 (베트남)
  └─ 1개월 성과 리뷰 + 2개월차 계획 조정
```

### 2개월차: 바이럴 가속

```
  ├─ 챌린지 결과 콘텐츠화 → 에브리타임/인스타 공유
  ├─ 공유 카드 기능 출시 → 자연 바이럴 트리거
  ├─ 에브리타임 5개 대학 이상 확장
  ├─ 월간 업종 트렌드 리포트 첫 발행
  └─ 유학생 대학 국제처 제휴 1곳 시도
```

### 3~4개월차: 그로스

```
  ├─ 인스타 광고 시작 (릴스 부스트)
  ├─ 유학생 앰배서더 프로그램 시작
  ├─ 초대 코드 시스템 출시
  ├─ "면접 빈출 주제 예측" 콘텐츠 시리즈 시작
  └─ 프리미엄 플랜 사전 알림 수집
```
